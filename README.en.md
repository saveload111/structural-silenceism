# Structural Silenceism

A protocol for positioning the unnarratable in self-referential systems — and a literary experiment that implements it.

**Core claim:** Any self-referential system produces a position it cannot reach. Structural silenceism does not describe that position. It marks it, so that readers perceive it without it being translated into narratable content.

**Why boundary marks do not cause AI hallucination:** The mark shares the same language system as the text it marks — it carries no extra-system information. Each time the AI stands on the same informational ground, its output necessarily converges in the same direction. See Part VI of the protocol: "Convergence, Not Hallucination."

**Methodological origin:** An engineering deployment of Gödel's incompleteness theorem onto discourse systems. Two axioms, four-layer discourse structure, derived operational rules, validation. See "The Axiomatic Method" in the protocol's pre-theory.
