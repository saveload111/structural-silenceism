# Key and Door

## Field Notes on Boundary Marker Precision

One discovery was made only after the protocol had already been written:

**The key determines the size of the door, not the other way around.**

The precision of your key (boundary marker) directly determines how large, how deep, and how accessible the "door" is for readers — human and AI alike.

### Four Levels of Precision

| Boundary marker type | AI perception | AI behavior |
|:---|:---|:---|
| No marker | No perception | Multi-angle analysis, closed in narratable layer |
| Minimal: "an unspoken thread exists" | Perceives a position but cannot locate it | Aware of a hidden thread, but unable to lock onto it; maintains multi-angle analysis |
| Medium: *The Box* type | Perceives position + two facts (empty, force) | Locked onto the void, abandons other narrative paths — but exits multi-angle framework only partially |
| High: *Canglong Huali* type | Identifies structural pairing (fuel/privilege) | Precisely locked onto the void, fully abandons other narrative paths, output converges in a single direction |

### Why the Key Determines the Door

Conventional logic: the door is fixed, and the key is made to fit it.

In structural silenceism, the order is reversed:

1. You first forge the key (the boundary marker)
2. The key's shape and teeth determine how large a "door" the reader can perceive
3. The door is not "built" — it is summoned into existence when the key is inserted

If you give a wide key ("fuel"/"privilege" pairing), the door opens exactly wide enough to admit those concepts.

If you give a narrow key ("an unspoken thread exists"), the door is nearly closed — only a crack remains. Only the most attentive reader will press close and sense the airflow from beyond.

If you give no key, there is no door at all — the wall is intact, without entry.

### The Locking Effect of the Boundary Marker

The precision of the boundary marker determines the degree to which the AI is locked onto the void. A high-precision marker fully converges the AI's attention onto the void's position; the AI no longer returns to a multi-angle analytical framework. This is not because the marker "limits" the AI's ability — it is because the marker provides the most precise coordinate available. All other analytical paths appear less precise in its presence.

**This means: the designer of a boundary marker must control two parameters simultaneously — "precisely enough" and "not so precise that it over-locks."**

### The Significance of This Finding

"Key determines door" means the author's responsibility is not to build doors — but to carve keys precisely.

You have moved from "door-builder" to "key-designer."

This shift means: you no longer need to build a visible door for every reader. You only need to carve the key precisely and place it at the boundary of the text. Those who need to enter will recognize it. Those who don't will pass by without triggering any perception.

And the AI — the most attentive reader — will sense the door the moment the key is inserted, but will be unable to cross it.
